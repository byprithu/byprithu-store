# byprithu Store — Setup Guide

Follow these steps in order. Takes about 10 minutes total.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 1 — Create your Firebase project
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Go to https://console.firebase.google.com
2. Click "Add project" → name it "byprithu-store" → Continue
3. Disable Google Analytics (not needed) → Create project

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 2 — Set up Firestore database
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. In your Firebase project → left sidebar → "Firestore Database"
2. Click "Create database"
3. Choose "Start in test mode" → Next
4. Pick any location (e.g. asia-south1 for India) → Enable

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 3 — Get your Firebase config keys
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. In Firebase → Project Settings (gear icon top left)
2. Scroll down to "Your apps" → click the </> Web icon
3. Register app — name it "byprithu-store" → Register
4. You'll see a config object like this:

   const firebaseConfig = {
     apiKey: "AIza...",
     authDomain: "byprithu-store.firebaseapp.com",
     projectId: "byprithu-store",
     storageBucket: "byprithu-store.appspot.com",
     messagingSenderId: "123456789",
     appId: "1:123456789:web:abc123"
   };

   Copy these values — you'll need them in Step 5.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 4 — Upload project to GitHub
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Go to https://github.com → sign in → New repository
2. Name it "byprithu-store" → Create repository
3. Upload all files from the byprithu-store/ folder you downloaded

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
STEP 5 — Deploy on Vercel + add Firebase keys
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Go to https://vercel.com → sign in with GitHub
2. Click "Add New Project" → select "byprithu-store"
3. Before clicking Deploy → click "Environment Variables"
4. Add these 6 variables (copy values from Step 3):

   Name                                   Value
   ─────────────────────────────────────────────────────────────
   NEXT_PUBLIC_FIREBASE_API_KEY           AIza...
   NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN       byprithu-store.firebaseapp.com
   NEXT_PUBLIC_FIREBASE_PROJECT_ID        byprithu-store
   NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET    byprithu-store.appspot.com
   NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID  123456789
   NEXT_PUBLIC_FIREBASE_APP_ID            1:123456789:web:abc123

5. Click Deploy → wait ~60 seconds
6. Your live URL appears! e.g. https://byprithu-store.vercel.app

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
DONE — Your permanent link is live!
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

- Seller dashboard: https://byprithu-store.vercel.app
- Customer form:    https://byprithu-store.vercel.app?poster=...  (generated by dashboard)
- Orders saved:     Firebase Firestore → visible in real time on all devices
- Update the site:  Edit files on GitHub → Vercel auto-deploys → same URL, never changes

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
UPDATING LATER
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

To update any code:
1. Go to your GitHub repo → click the file → pencil (edit) icon
2. Make changes → click "Commit changes"
3. Vercel auto-deploys in ~60 seconds
4. URL stays exactly the same forever

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
CUSTOM DOMAIN (optional)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Vercel dashboard → your project → Settings → Domains
2. Add domain → follow DNS instructions
